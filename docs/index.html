Job Notice Monitor

Tracks configured websites and PDFs, removes expired notices where possible, and lets each user filter results for their own profile.

Your filters

These filters run in your browser. Resume/skill keywords are stored locally and are not committed to GitHub.

Add a website

Static GitHub Pages cannot permanently add a new scrape target by itself. This form saves the request in your browser and creates YAML you can paste into `config/sources.yaml`, then commit to GitHub.

Scrape diagnostics

A source can be configured but absent from live results if it blocked scraping, returned no matching links, or had no live notices after filtering.
