          ℹ️ How this works:


          The scraper runs every 6 hours on GitHub Actions (free for public repos).

          It reads config/sources.yaml, fetches each URL, and writes results to docs/data/jobs.json.


          To add or remove sources, edit config/sources.yaml in your GitHub repo and commit the change.
          The next scheduled run (within 6 hours) will pick it up automatically.
